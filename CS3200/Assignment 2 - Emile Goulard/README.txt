DO NOT RUN: 'A2_RNGClass.m' (it is meant for 'A2_RandomNumberGenerator.m')

TO RUN "A2_RandomNumberGenerator"

	1) Open "A2_RandomNumberGenerator.m" in MATLAB
	2) Press the Dropdown arrow underneath the RUN button and enter an integer parameter. 
		This will represent the size n for generating however many random numbers you wish.

TO RUN "A2_MonteCarloMethod"

	1) Open "A2_MonteCarloMethod.m" in MATLAB
	2) Press the Dropdown arrow underneath the RUN button and enter an integer parameter first and then a boolean parameter second. 
		This 1st parameter will represent the size n for generating however many random numbers you wish.
		The 2nd parameter will dictate whether to run My RNG formula or the Built-In RNG formula to compute the Monte Carlo approach.
		(Set to true for My RNG formula) (Set to false for Built-In RNG formula)
